# Taller I - Ejemplos de Java

## Descripción

Este repositorio contiene una serie de ejemplos prácticos en Java que muestran el uso de diferentes características del lenguaje. Cada programa permite la interacción con el usuario mediante la clase `Scanner` y demuestra una funcionalidad específica de Java.

## Contenido del proyecto

### 1. VarargsExample.java

Demuestra el uso de **Varargs (`int...`)**, permitiendo sumar una cantidad variable de números ingresados por el usuario.

**Funciones:**

* Solicita la cantidad de números.
* Permite ingresar los valores.
* Calcula y muestra la suma total utilizando un método con Varargs.

---

### 2. StringJoinExample.java

Ejemplo del método `String.join()` para unir varias cadenas de texto.

**Funciones:**

* Solicita tres nombres.
* Une los nombres separados por comas.
* Muestra el resultado en pantalla.

---

### 3. IsBlankExample.java

Ejemplo del método `isBlank()` de la clase `String`.

**Funciones:**

* Solicita un texto al usuario.
* Verifica si la cadena está vacía o contiene únicamente espacios.
* Informa si la cadena tiene contenido o está vacía.

---

### 4. RequireNonNullExample.java

Ejemplo del método `Objects.requireNonNull()`.

**Funciones:**

* Solicita un nombre.
* Valida que el valor no sea nulo.
* Lanza una excepción si el usuario no ingresa un nombre válido.

---

### 5. TryWithResourcesExample.java

Ejemplo de la estructura **Try-With-Resources** para manejar archivos de forma segura.

**Funciones:**

* Solicita el nombre de un archivo.
* Intenta abrirlo y leer su primera línea.
* Maneja automáticamente el cierre del recurso.
* Captura posibles errores de lectura.

---

## Requisitos

* Java JDK 8 o superior.
* Visual Studio Code, IntelliJ IDEA o cualquier IDE compatible con Java.

## Cómo ejecutar

1. Clone este repositorio.
2. Abra el proyecto en su IDE favorito.
3. Compile el archivo Java que desee ejecutar.
4. Ejecute la clase correspondiente.
5. Siga las instrucciones que aparecen en la consola.
